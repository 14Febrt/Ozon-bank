# Иконка приложения

Положи сюда файл `icon.png`:

- размер **1024×1024 px**
- квадратный, **без прозрачности** (для iOS обязательно — мы её всё равно вырежем флагом `remove_alpha_ios`)
- PNG

После этого запусти в корне проекта:

```
flutter pub get
dart run flutter_launcher_icons
```

Это сгенерирует иконки нужных размеров для iOS / Android / Web.
Затем закоммить и пуш — сборка на Codemagic подхватит новую иконку.

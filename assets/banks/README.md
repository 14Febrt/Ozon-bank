# Логотипы банков

Положи сюда PNG-файлы (или JPG) с такими именами:

- `sber.png` — Сбербанк
- `tbank.png` — Т-Банк
- `ozon.png` — Ozon Банк
- `alfa.png` — Альфа-Банк

Рекомендуемый размер: 120×120 px, прозрачный фон (PNG).

Если файла нет — приложение автоматически загрузит логотип из интернета
(Clearbit Logo API), а если и сети нет — нарисует упрощённый плейсхолдер.

После добавления файлов запусти `flutter pub get` и пересобери приложение.
